Windows - Win Server 2016 伺服器安裝及設定

Windows Server 2016 是 Windows 10 的服器版本，並具備多樣功能，例如 AD、 IIS、 Hyper-V、 DNS 、路由等， 比起 Linux 作業系統安裝及設定相對容易，作為練習評估是很好的上手的系統。

1. Windows 2016 介紹

Windows Server 2016 為新一代的雲端作業平台，有簡易上手的的GUI，使 IT 管理人員容易管理，且微軟官方有提供 Windows Server 2016 評估版 ( 180 天 )，官方網站下載 ISO 評估版連結請點我。

2. 使用 VMware 安裝 Windows 2016 Server

囿於只是評估練習 Win 2016 ， 不需要特別找一台實體機器來安裝，可以透過 VMware 虛擬化系列來安裝(此 Lab 使用 VMware Workstation Player )

  1. VMware 選擇創建新的虛擬機器。

  1. 選擇下載的 Windows Server 2016 ISO 檔案

NOTE
VMware 會自動偵測光碟片，如出現 Win Server 2016 被偵測到就可以使用簡易安裝模式。
  1. 選擇 Standard 版本安裝，並創建 student 使用者。

NOTE

Windows 2016 版本系列差異：

  • Datacenter ： 完整旗艦版，可以執行高度虛擬化，另有超融合架構，且虛擬機器授權數量無上限。
  • Standard : 標準版，僅提供 2 台虛擬機器授權。
  • Core : 不具備圖形化介面，只能透過遠端伺服器管理員來管理。
  1. 產品金鑰提示，點擊 Yes 使用評估版。

  1. 設定虛擬機名稱及存放路徑，依個人喜好決定。

  1. 虛擬硬碟空間使用預設 60 Gib ，並選擇單一檔案存放。

TIP
單一檔案存放好處讀寫效能較高，預設 VMware 使用分割多的檔案存放。
  1. 選擇客制化硬體。

  1. 移除掉無用的介面卡

  1. 點擊 Finish 會開始自動進行安裝

  1. Windows 安裝完，VMware 會自動安裝相關套件，使虛擬化更方便使用。

3. Windows 2016 Server 基本設定

Windows 2016 Sever 安裝十分容易，接下來設定一些常用配置。

3.1. 編輯電腦名稱

  1. 伺服器管理員面板 -> 本機伺服器 -> 電腦名稱 ( WIN-xxxx ) -> 系統內容的變更 -> 編輯電腦名稱 -> 確定。

  1. 變更完電腦名稱，系統會要求重新啟動才會生效。

3.2. 設定靜態 IP

Windows Server 通常都會使用靜態 IP ，使得管理及維護方便。

  1. 伺服器管理員面板 -> 本機伺服器 -> Eth0 -> 右鍵內容 -> 選擇 IPv4 -> 點擊內容。

  1. 編輯 IP 組態及 DNS。

3.3. 設定 Administrator 密碼並且開機使用

Windows 預設會啟用一般 User 並且使用無密碼登入，不過該方式對於 Server 風險較大，故設定管理員密碼，並且要求一定鑰輸入密碼才能登入相對安全。

  1. 左下角 Windows 圖示點擊右鍵 -> 命令提示字元 ( 系統管理員 ) 。

  1. 選擇是允許 App 變更裝置。

  1. 設定 admininstrator 密碼 ( 可自定義，範例使用 P@ssw0rd )。
1

net user administrator P@ssw0rd

  1. 開啟電腦登入必須使用帳號及密碼。

  1. 登出 student 帳號選擇 Administrator 就會需要輸入密碼。

也可以將 student 停用或是也設定密碼，提高安全性。

3.4. 防火牆開放 ICMP 協定

  1. 同上開啟命令提示字元 ( 系統管理員 )，並輸入下方指令。
1

netsh advfirewall firewall add rule name="ICMP" protocol=icmpv4:8,any dir=in action=allow

  1. 使用任一同網段機器 Ping Win Server 2016 ，將可以看見封包回應。

4. 小結

Windows Server 在金融證券業使用率有一定的市場，除了 Linux 技巧必須要掌握， 基礎的 Windows 也要稍微瞭解。
