Linux - RedHat 8 圖形介面與終端介面切換

啟用圖形介面 CTRL + ALT + F2

Note
預設登入之後就是進入到F2中

圖形介面更換使用者 CTRL + ALT + F1

在不登入原本使用者情況下，要多人多工增加使用，可以透過F1登入

Note
一旦登入後會另外尋找沒有使用過的F3介面

圖形介面切換成原使用者 CTRL + ALT + F2

圖形介面切換新使用者 CTRL + ALT + F3

使用終端介面登入 CTRL + ALT + F4 ~ F7

Note
最多到F7

小結

Linux 完全可以實現多人多工操作，不會像是 Windows 當下只能容許一個使用者登入，其他會自動被登出 ～
