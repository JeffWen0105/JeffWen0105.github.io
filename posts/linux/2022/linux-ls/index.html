Linux - ls 檢視檔案與目錄

LS 是 Linux(Unix Like) 最常使用命令之一，用來檢視檔案與目錄的狀態。

Linux 常用指令 - ls 檢視檔案與目錄

ls 指令參數說明

  • 參數：
    • -a：列出所有檔案，包含隱藏檔( 開頭為 . 的檔案)
    • -h：可將檔案容易以易讀式呈現(EX：GB、 KB、…)
    • -l：將資料以列表形式呈現，並且包含檔案細部資料(屬性、權限、時間…)
    • -r：反向排序(原先是從a->z變成z->a)
    • -R：將目錄下的子目錄檔案一起列出(白話：整個資料夾檔案列出)
    • -S：用檔案大小排序
    • -t：用時間排序

ls -l 欄位詳細說明

  1. File Type
    • 一般檔案 : -
    • 目錄 : d
    • 軟連結 : l
    • 設備檔 : b
    • 設備檔 : c
    • Unix Domain Socket : s
    • pipline : p
  2. 一般權限(Owner、Group、Other)
  3. ACL 權限 （開啟呈現 + 號)
  4. 如果 File Type 為 d (目錄) ，表示該目錄底下有多少目錄 ; 如果不為 d (目錄)，表示有多少個 Hard Link 數量
  5. 檔案或目錄的 Owner
  6. 檔案或目錄的 Group
  7. 檔案大小(byte)
  8. 時間 ( 顯示為 Modify Time)
    Linux 四種時間
    時間欄位 說明
    Access Time 最後一次讀取的時間(cat、more、less..等)
    Modify Time 最後一次修改檔案內容時間
    Change Time 最後一次修改檔案權限或是擁有者的時間
    Birth Time 檔案創建時間

  9. 檔名
