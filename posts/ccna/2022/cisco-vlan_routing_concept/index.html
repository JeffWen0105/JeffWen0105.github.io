在不同的 VLAN 如果要互相溝通需要透過路由協助轉發至對應的 VLAN 內。

1. VLAN 路由方式

  1. Legacy Inter VLAN routing
  2. Router on a Stick
  3. Layer 3 Switch

1.1. Legacy Inter VLAN routing

傳統的方式，囿於需要額外佔用兩個 Router 的實體 port ，僅適合迷你形網路。

Note
所有的傳輸都是透過 Access mode，並沒有使用 Trunk Mode。

1.2. Router on a Stick

透過一個 Router 中分成多個虛擬子介面來達成邏輯介面，並透過此方式取代 Legacy Inter VLAN routing 。

1.3. Layer 3 Switch

透過 L3 Switch 是最容易的方式，不需藉由 Router 也能達成。

2. Router on a Stick 設定簡述

Router 開啟子介面，將每一個 VLAN 切成一個個虛擬介面並賦予 IP 及開啟 802.1Q。

  int g0/0.10
    encapsulation dot1Q 10
    ip address 192.168.10.1 255.255.255.0
  int g0/0.30
    encapsulation dot1Q 30
    ip address 192.168.20.1 255.255.255.0
  int g0/0
    no shut

3. Layer 3 Switch 設定簡述

L3 Switch 很單純僅在 Vlan 上賦予IP，最後開啟 Routing 就會自動產生路由。

conf t
int Vlan10
 ip address 192.168.10.254 255.255.255.0
 no shut
int Vlan20
 ip address 192.168.20.254 255.255.255.0 
 no shut
ip routing

4. 小結

一般中型網路大多數使用 Router on a Stick 方式，不過預算充足可以直接考慮 L3 Switch ，設定管理相對容易。
