在不同的 VLAN 如果要互相溝通需要透過路由 ( 使用 L3 Switch 取代一般的路由器 )協助轉發至對應的 VLAN 內。

先前介紹的三種路由方式，路由速度最快設定步驟容易的方式，缺點就是 L3 Switch 價格不斐。

1. 設定步驟

  1. 所有的 Switch 將所有 Port 端口都貼上 Vlan 標籤。
  2. 創建 SVI Vlan 介面。
  3. 啟動 Ip Routing 。
NOTE
如果沒有跨越 Switch 不需要啟用 Trunk Mode。

1.1 L3 Switch 外接路由器步驟

如果 L3 Switch 要連到外部路由器，僅需將原本的 L2 介面切換至 L3 並賦予 IP 即可。

2
3
4

int g0/0
    no switchport
    ip address 192.168.100.1 255.255.255.0
    no shut

    switchport trunk native vlan 99
    switchport trunk encapsulation dot1q

3. 小結

在預算充足下，使用 L3 Switch 是最好的方案，不只在處理封包或是設定上都特別出色。
