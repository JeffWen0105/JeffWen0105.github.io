練習環境
可以使用 HowHow 的創建 Lab 練習環境，來操作多台機器練習 Ansible，使用方式請參閱 HowHow 網站。

題目

**在控制主機上 : **

  • 創建 /home/student/ansible/hwreport.yml 檔案。
  • 在 dev 組執行，並輸出 /root/hwreport.txt 報告。
  • 報告範本如下 :
1
2
3
4
5
6

nodename =  ansible_facts.nodename
memory =  ansible_facts.memtotal_mb
bios_version = ansible_facts.bios_version
vda_size = ansible_facts.devices.vda.size
vdb_size = ansible_facts.devices.vda.size
vdc_size = ansible_facts.devices.vdc.size
提示
  • 考試期間是可以查閱 Ansible DOC 文件。
  • Ansible 模組要多使用 ansible-doc 指令內 example 範例來複製修改。

---
- name: Get hwreport info
  hosts: dev
  tasks:
    - name: create report file
      template:
        src: hwreport.txt.j2
        dest: /root/hwreport.txt
