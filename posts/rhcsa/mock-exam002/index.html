題目

設定 ServerA :

  • BaseOS URL : http://mirror01.idc.hinet.net/rockylinux/8.5/BaseOS/x86_64/os/
  • BaseOS 名稱 : BaseOS
  • BaseOS 描述 : ex200-baseos
  • GPGKey : http://mirror01.idc.hinet.net/rockylinux/RPM-GPG-KEY-rockyofficial
  • AppStream URL : http://mirror01.idc.hinet.net/rockylinux/8.5/AppStream/x86_64/os/
  • AppStream 名稱 : AppStream
  • AppStream 描述 : ex200-appstream
  • GPGKey : http://mirror01.idc.hinet.net/rockylinux/RPM-GPG-KEY-rockyofficial
提示
  • 操作前刪除所有的倉庫清單 rm -f /etc/yum.repos.d/* 。
  • 配置完使用 yum install zsh ，顯示是否下載同意畫面表示 BaseOS 配置正確。
  • 配置完使用 yum install httpd ，顯示是否下載同意畫面表示 AppStream 配置正確。
  • 不需執行 rht-vmctl reset servera 重置機器，不會影響後續的 LAB。

參考作法

  1. 使用 nmcli 工具設定 。
  2. 使用 nmtui 工具設定 (圖形化介面較簡單)。
  3. man 相關手冊 -> NetworkManager。
